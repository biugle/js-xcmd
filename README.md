# js-xcmd

## Install

```bash
npm i js-xcmd -g
```

## Use

```bash
xcmd -h

const {
  isDirExistResult,
  isDirExist,
  copyDir,
  copyFile,
  deleteDir,
  deleteFile,
  addDir,
  addFile,
  renameDir,
  renameFile,
  getFileName,
  getFileSize,
  getFileExt,
  getFileContent,
  setFileContent,
  cmd
} = require('js-xcmd');
```

## Test Package.json.bin.xxx

```bash
npm link
npm run bind
npm unlink
npm run unbind
```
